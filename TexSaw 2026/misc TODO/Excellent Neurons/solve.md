Writeup:
Нейронная сеть в Excel имеет архитектуру Input(30) → ReLU → Hidden1(60) → ReLU → Hidden2(1) → ReLU → Output(4) → Sigmoid. Вывод считается FLAG если F>0.5, L<0.5, A>0.5, G<0.5.
Ключевое наблюдение: матрица весов W1 разреженная — каждый нейрон зависит ровно от одного входного символа с весом ±1. Все веса W2 отрицательные, поэтому для выхода FLAG нужно чтобы все активации a1 были равны нулю.
Из условий a1[i] = 0 для каждого нейрона однозначно получается допустимый диапазон входного значения (ASCII/127). Для каждой позиции 0–21 диапазон сужается до одного конкретного символа — так восстанавливается флаг целиком.
