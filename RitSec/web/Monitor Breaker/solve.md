
Writeup:

Веб-приложение представляет собой панель администратора, где пути к рабочим инструментам обфусцированы через MD5-хеши обычных чисел. Скрытый интерфейс "Network Ping Tool" (доступный по пути /_sys/cfcd208495d565ef66e7dff9f98764da, что является MD5 от 0) уязвим к OS Command Injection через параметр target.

Ключевое наблюдение: Разработчик попытался защититься от внедрения команд, добавив клиентский JavaScript-фильтр, который удаляет все латинские буквы (/[a-zA-Z]/g) прямо во время ввода в HTML-форму. При этом санитаризация данных на стороне сервера полностью отсутствует.

Обход фильтра (Bypass) выполняется элементарно: отправкой прямого POST-запроса в обход UI с помощью Fetch API в консоли разработчика. Инъекция пейлоада 127.0.0.1; ls -la позволяет прочитать директорию и обнаружить файл flag-9d444ad0f475b52e79a1713f25646dce.txt, а последующая отправка команды 127.0.0.1; cat [имя_файла] выводит содержимое файла в JSON-ответе сервера.
