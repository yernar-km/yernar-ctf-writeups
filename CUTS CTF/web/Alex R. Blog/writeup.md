Описание: Веб-приложение на Flask, стилизованное под личный блог программиста из 2002 года.

Шаг 1: Разведка (Recon)
Изучая посты в блоге, находим прямую подсказку в посте «Призрак в машине»: автор упоминает robots.txt и историю git.
Проверка /robots.txt раскрывает два скрытых пути:

/admin-1337 — панель авторизации (стандартные пароли и SQLi не работают).

/vault — секретное хранилище (выдает ошибку 403: неверный токен).

Шаг 2: Обнаружение уязвимости (Source Code Disclosure)
Следуя второй подсказке из блога, проверяем наличие открытой директории .git. Запрос к /.git/config возвращает содержимое файла конфигурации, подтверждая уязвимость.

Шаг 3: Эксплуатация
С помощью утилиты git-dumper (или wget) полностью скачиваем репозиторий сайта на локальную машину:

Bash
git-dumper http://challs.cuts.uz:5021/ site_dump
cd site_dump
git checkout .
Шаг 4: Анализ исходного кода и получение флага
Читаем восстановленный файл app.py и находим захардкоженные секреты, включая учетные данные админа (admin : r4d10h34d) и токен для /vault (gh0st_1n_th3_m4ch1n3).
Здесь же в открытом виде лежит сам флаг, который передается через переменные окружения.

Флаг:

Уязвимость: Exposed .git directory (CWE-526: Information Exposure Through Environmental Variables / Exposed Source Control).
